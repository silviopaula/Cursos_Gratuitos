**Organização**: *Softblue*                            
**Carga Horária**: *8 horas*                             
**Idioma**: Português  
**Certificado**: *Sim*                                  
**Descrição**: *A orientação a objetos é o paradigma adotado pelas linguagens de programação modernas. Mas só conhecer os seus conceitos nem sempre é o suficiente.
